Making password reset systems for the betterment of humanity. And also cute fluffy organisms, because they are cute and fluffy and fun to snuggle. Unless they're cute and fluffy and bitey, which would be negative. How much of this description is the site going to show? Consider it stress testing, really. After all, some people like to write a lot and some people only write a little -- we have to assume that this system was more designed for the likes of those who only like to write a little about what they're doing and how it will affect them / the company / the city / state / country / continent / planet / solar system / galaxy / universe. I think the only thing bigger than the concept of the universe is the concept of a multiverse, where an untold number of universes exist simultaneously and technically within the same space, but not the same time -- it's weird. In fact, here's the first few paragraphs from wikipedia about the concept of multiverses:

The multiverse (or metaverse[1]) is a hypothetical group of multiple universes including the universe in which humans live. Together, these universes comprise everything that exists: the entirety of space, time, matter, energy, the physical laws and the constants that describe them.[2][3][4][5] The different universes within the multiverse are called the "parallel universes", "other universes", or "alternative universes".[6][7][8]

Doesn't that sound interesting? I'm really starting to think that this field should have a length limit. Maybe eventually I should talk about the actual project? I've got 51 minutes to continue typing, typing, typing away... I hope I eventually hit a character limit because otherwise I'm going to be here for a very long time.
